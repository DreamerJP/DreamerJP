## <img src="https://icongr.am/feather/terminal.svg?size=26&color=20B2AA" valign="middle" /> Automação & Scripts

### <img src="https://icongr.am/feather/download-cloud.svg?size=20&color=20B2AA" valign="middle" /> [Downloader Automático](Downloader/Atual/)

> _Acelerador de downloads com segmentação de arquivos._

![Python](https://img.shields.io/badge/python-3670A0?style=flat-square&logo=python&logoColor=ffdd54)
![Speed](https://img.shields.io/badge/Focus-High_Speed-red?style=flat-square)

Ferramenta avançada desenvolvida para **superar limitações de velocidade de servidor**. O script divide o arquivo alvo em múltiplas partes e realiza o download simultâneo de cada uma delas, maximizando a banda disponível.

- <img src="https://icongr.am/feather/code.svg?size=14&color=4682B4" valign="middle" /> **Tecnologia:** Segmentação de downloads em partes paralelas.
- <img src="https://icongr.am/feather/box.svg?size=14&color=4682B4" valign="middle" /> **Integração:** Python + PyInstaller (Compatível com Windows).
- <img src="https://icongr.am/feather/activity.svg?size=14&color=4682B4" valign="middle" /> **Recursos:** Histórico JSON, retries automáticos e unificação pós-download.
